Detailed Description

Stores a copy of the gradients of a generative function.

For use in accumulating gradients with Trace::gradients. Typically there will be multiple GradientAccumulators for a given Module.

Also see Module.

Not thread-safe. Each thread should have its own GradientAccumulator.
